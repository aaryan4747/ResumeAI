An enterprise-grade, full-stack recruitment platform designed to automate and enhance high-volume candidate screening. Powered by Java Spring Boot, a custom Glassmorphism UI, and the Google Gemini 2.0 API, this system moves beyond simple keyword matching. It features asynchronous document ingestion (PDF, DOCX, TXT), JWT-secured endpoints, and a Vector-based Semantic Search engine that mathematically ranks applicants based on deep conceptual understanding rather than strict vocabulary. Manage your talent pool with real-time analytics, bulk export tools, and dynamic AI-weight tuning for precision hiring.
